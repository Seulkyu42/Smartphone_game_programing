
# 게임제목 : MonsterSurvival

## 게임컨셉
기존 게임 "레이디 버그"를 모티브로
픽셀아트 (도트) 느낌의 일방진행 서바이벌 게임


## 게임 플로우 
자이로 센서를 이용한 플레이어 조종과
랜덤으로 생성되는 아이템을 통해 몰려오는 적들로부터 살아남기


# 개발범위

## 조작 
  * 로비 버튼조작 
  * 자이로 센서의 기울기로 플레이어 이동 
  
##  배경
  * 심리스 이미지로 자연스러운 스크롤링
## 플레이어 스텟 
  * 몬스터를 처치하고 얻은 코인으로 성장
  * 체력 (기존 3칸 / 최대 5칸)
  * 이동속도 (최대 20퍼센트)

## 개발일정 

|기간|내용|
|:---|---:|
|1주차| 이미지 수집 및 부족 이미지 제작|

|7주차| 아이템 구현|
